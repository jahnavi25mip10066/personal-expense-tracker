# Personal Expense Tracker

##  Overview
The Personal Expense Tracker is a simple Python-based console application that helps users record, update, view, and delete their daily expenses.  
The project stores data temporarily in a Python list (in-memory), and everything resets once the program stops.  
It is designed for beginners and follows a modular structure with multiple Python files.

##  Features
- Add new expenses  
- View all added expenses  
- Update an existing expense  
- Delete an expense  
- User-friendly menu  
- Data stored only during runtime (no database used)  
- Uses 5 Python modules (school/college-level)

##  Technologies Used
- Python 3  
- Basic functions  
- Classes & objects  
- Lists (for storage)  
- Modular programming (multiple .py files)
